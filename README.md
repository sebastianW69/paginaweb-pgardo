# Proyecto Marketplace Textil

Este proyecto es un prototipo de un marketplace textil llamado **TextilBog**, diseñado para ofrecer moda exclusiva, artesanal y sostenible en Bogotá. A continuación se detallan los componentes y la estructura del proyecto.

## Estructura del Proyecto

```
pagina-adelanto
├── css
│   └── estilo.css          # Contiene los estilos para toda la aplicación.
├── js
│   └── mecanicas.js        # Contiene la funcionalidad JavaScript para la aplicación.
├── imagenes
│   ├── inicio.png          # Imagen utilizada como visual principal en la página de inicio.
│   ├── campesino.png       # Imagen del producto "Blusa campesina - Algodón".
│   ├── lino.png            # Imagen del producto "Camisa lino stretch".
│   ├── vestido.png         # Imagen del producto "Vestido verano".
│   ├── cuero.png           # Imagen del producto "Chaqueta cuero legítimo".
│   └── bolso.png           # Imagen del producto "Bolso artesanal en fique".
├── paginas
│   ├── iniciar-sesion.html  # Página de inicio de sesión para usuarios.
│   ├── categoria.html       # Página de categorías de productos.
│   ├── vender.html          # Página para que los usuarios vendan sus productos.
│   └── mi-cuenta.html       # Página de cuenta de usuario para gestionar perfiles y configuraciones.
├── index.html               # Punto de entrada principal de la aplicación.
└── README.md                # Documentación del proyecto.
```

## Instalación

1. Clona este repositorio en tu máquina local.
2. Abre el archivo `index.html` en tu navegador para ver la aplicación en acción.

## Funcionalidades

- **Navegación**: Los usuarios pueden navegar a través de diferentes categorías de productos.
- **Carrito de Compras**: Los usuarios pueden agregar productos a su carrito y proceder al pago.
- **Comentarios**: Los usuarios pueden dejar comentarios sobre la página y los productos.
- **Módulos de Usuario**: Los usuarios pueden iniciar sesión y gestionar su cuenta.
